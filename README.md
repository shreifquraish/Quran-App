# القرآن الكريم

تطبيق Flutter احترافي لقراءة واستماع القرآن الكريم باللغة العربية.

## المميزات

- **قراءة بدون إنترنت** — تحميل المصحف (حفص) من [Quranpedia API](https://api.quranpedia.net/v1) عند أول تشغيل
- **بحث في القرآن** — بحث محلي في نص المصحف بدون إنترنت
- **علامات مرجعية** — حفظ الآيات والعودة إليها لاحقاً
- **تفسير الآيات** — من Quranpedia مع تخزين مؤقت للاستخدام لاحقاً
- **استماع مع تنزيل اختياري** — 9 قراء من [MP3Quran API](https://mp3quran.net/api/v3)
- **إذاعة مباشرة** — بث تلاوات القراء المختارين
- **واجهة عربية RTL** — تصميم داكن مريح للعين مع دعم الوضع الفاتح

## القراء المتاحون

1. عبد الباسط عبد الصمد
2. ياسر الدوسري
3. محمود خليل الحصري
4. ماهر المعيقلي
5. مشاري بن راشد العفاسي
6. أحمد بن علي العجمي
7. عبد الرحمن السديس
8. محمد صديق المنشاوي
9. مصطفى إسماعيل

## التشغيل

```bash
flutter pub get
flutter run
```

> **ملاحظة:** عند أول تشغيل يحتاج التطبيق إنترنت لتحميل نص المصحف. بعد ذلك القراءة تعمل بدون إنترنت. التلاوات تُنزَّل حسب اختيار المستخدم.
