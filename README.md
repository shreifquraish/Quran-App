| # | القارئ |

## ✨ مميزات التطبيق

### 🎙️ الاستماع (التلاوات)
- قائمة بأشهر القراء
- تشغيل مباشر عبر الإنترنت ▶️
- تحميل السور للاستماع بدون نت ☁️
- تحميل المصحف كاملاً لشيخ معين
- مشغل مصغر في أسفل الشاشة

### 📻 الإذاعات (البث المباشر)
- استمع للقنوات القرآنية مباشرة
- بدون توقف – بث حي 24 ساعة

### 📖 القراءة (المصحف)
- مصحف كامل برسم عثماني
- حفظ تلقائي لآخر موضع قراءة
- عرض رقم الصفحة والجزء والحزب

### 📚 التفسير
- 3 كتب تفسير متاحة (الميسر، الجلالين، السعدي)
- تفسير كامل لكل سورة
- إمكانية التبديل بين كتب التفسير بسهولة

### ⚙️ إضافات
- البحث والعلامات
- معرفة حالة تحميل المصحف
- التواصل المباشر مع المطور

## 🚀 كيفية الاستخدام

1. **الاستماع:** اختر قارئاً ← اختر سورة ← اضغط تشغيل أو تحميل
2. **القراءة:** اختر السورة ← اقرأ المصحف ← يتذكر آخر صفحة لك تلقائياً
3. **التفسير:** اختر التفسير المناسب ← اختر السورة ← اقرأ التفسير
4. **الإذاعة:** اختر قناة قرآنية ← استمع مباشرة

🔁 يمكنك التبديل بين الأقسام من خلال شريط التبويب السفلي.
